## הגדרת המאגר

נדרשים שני secrets ב-GitHub repository:
- `GEMINI_API_KEY` — מפתח Gemini API מ-Google AI Studio
- `TWITTER_API_KEY` — מפתח מ-twitterapi.io

לאחר הגדרת ה-secrets:
1. הפעל GitHub Pages על branch `main` בהגדרות הריפו
2. הרץ ידנית `workflow_dispatch` עם `review_type` רצוי כדי לבדוק
3. ה-cron יפעל אוטומטית בהמשך
